## Introduction

The OSM API serves as an interface to retrieve GeoJSON features for a specified location using a geolocation bounding box.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## Usage
### Example Request:
```bash
GET localhost:3000/features?bbox=-122.4055,37.784,-122.4045,37.785
```

## Test

```bash
$ npm run test
```
